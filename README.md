# Spring CRUD Application

## Overview

This is a Spring Boot CRUD (Create, Read, Update, Delete) application that manages user data. It utilizes Spring Data JPA to interact with a MySQL database.

## Features

- Add new users
- Retrieve all users
- Retrieve a user by ID
- Update user details
- Delete a user

## Technologies Used

- Java 21
- Spring Boot 3.3.4
- Spring Data JPA
- MySQL
- Maven
